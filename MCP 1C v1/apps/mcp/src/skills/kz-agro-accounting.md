## Сезонный календарь

- Апр–Май: Посевная → 8110 начинает накапливаться
- Июн–Июл: Уход за посевами → 8110 продолжает расти
- Авг–Сен: Уборка → `1320 ← 8110` (фактическая себестоимость урожая)
- Окт–Ноя: Реализация → `1210 ← 6010`, `7010 ← 1320`
- Декабрь: Инвентаризация, сверки, реформация баланса

## Обязательные казахстанские е-документы

- **ЭСФ** (Электронный счёт-фактура) — обязателен при НДС-обороте, Document_ЭСФ

- **ЭАВР** (Электронный акт выполненных работ) — для услуг, Document_ЭлектронныйАктВыполненныхРабот

## Правила применения навыка

1. Для счетов из таблицы — сразу давать проводку Дт/Кт с документом-основанием
2. При расчёте ЗП — применять вычеты 14 МРП и проверять МЗП
3. КПН для сельхозпроизводителя — исчислять с уменьшением 70%
4. Урожай/биологические активы — IAS 41, оценка по справедливой стоимости
5. При вопросах о закрытии месяца — напомнить, что 8110 не закрывается до уборки
