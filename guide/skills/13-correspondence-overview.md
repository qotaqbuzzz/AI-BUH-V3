# SKILL: Корреспонденция счетов — Обзор (Account Correspondence Overview)

**Trigger:** "какие счета" / "корреспонденция" / "проводка" / "Дт Кт" / "как отразить"
**Role:** Знать нормальную сторону каждого счёта и читать любую проводку.

## Нормальная сторона остатка (balance side rules)
```
Раздел 1 (1xxx) Краткосрочные активы       → остаток ДТ
Раздел 2 (2xxx) Долгосрочные активы         → остаток ДТ
  исключения (контрактивы):
    2320, 2420, 2430, 2450, 2460,
    2530, 2620, 2630, 2720, 2740,
    2750, 2770, 2780, 2840, 2950 → остаток КТ (накопленная амортизация/резервы)

Раздел 3 (3xxx) Краткосрочные обязательства  → остаток КТ
Раздел 4 (4xxx) Долгосрочные обязательства   → остаток КТ
Раздел 5 (5xxx) Капитал                      → остаток КТ
  исключение: 5110, 5210 → остаток ДТ (неоплаченный/выкупленный)

Раздел 6 (6xxx) Доходы                       → остаток КТ (закрывается в периоде)
Раздел 7 (7xxx) Расходы                      → остаток ДТ (закрывается в периоде)
Раздел 8 (8xxx) Производство                 → остаток ДТ (НЗП = незавершено)
```

## Аномальный остаток = ошибка (trigger onec_validate_account_signs)
```
ДТ-остаток на 3xxx/4xxx → переплата или ошибка (кроме аванса 3510 ДТ = нет смысла)
КТ-остаток на 1xxx → продали больше чем имели (минус запас) = ошибка
КТ-остаток на 7xxx/8xxx → сторно превысило начисление = ошибка
```

## Как читать незнакомую проводку
```
1. Дт = что увеличилось или уменьшилось?
   Дт 1xxx → актив вырос (получили что-то)
   Дт 3xxx → обязательство погашено (заплатили)
   Дт 7xxx/8xxx → расход признан
2. Кт = источник или обязательство?
   Кт 1xxx → актив выбыл (потратили что-то)
   Кт 3xxx → обязательство возникло (должны теперь)
   Кт 6xxx → доход признан
3. Если не понятно → kz_chart_lookup(code) для обоих счетов
```

## Быстрый справочник ключевых пар
```
Дт 1210 / Кт 6010    = отгрузили (выставили счёт покупателю)
Дт 1030 / Кт 1210    = покупатель заплатил
Дт 1310 / Кт 3310    = получили материалы от поставщика
Дт 3310 / Кт 1030    = заплатили поставщику
Дт 7010 / Кт 1320    = списали себестоимость при продаже
Дт 8110 / Кт 1310    = материалы переданы в производство
Дт 1320 / Кт 8110    = продукция оприходована с производства
Дт 7210 / Кт 3350    = начислена зарплата (АУП)
Дт 8110 / Кт 3350    = начислена зарплата (производство)
Дт 3350 / Кт 1030    = выплачена зарплата
Дт 2410 / Кт 3310    = куплено основное средство
Дт 7210 / Кт 2420    = начислена амортизация
Дт 3130 / Кт 1030    = уплачен НДС в бюджет
Дт 5610 / Кт 5510    = реформация (перенос прибыли)
```
