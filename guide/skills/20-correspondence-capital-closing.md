**Trigger:** счета 5xxx/6xxx↔7xxx закрытие / реформация / дивиденды / курс. разница

## Накопленная прибыль и реформация (5610/5510)
| Дт | Кт | Операция | Когда |
|---|---|---|---|
| 6010..6xxx | 5610 | Доходы закрываются в прибыль (ЗакрытиеМесяца) | Ежемесячно |
| 5610 | 7010..7xxx | Расходы закрываются из прибыли | Ежемесячно |
| 5610 | 5510 | Реформация: прибыль → накопленная (год. итог) | 31 декабря |
| 5510 | 5610 | Реформация: убыток покрыт из накопленной | 31 декабря |

```
После реформации: 5610 = 0, 5510 растёт год за годом
Если 5510 < 0 → накопленный убыток → риск неплатёжеспособности
```

## Закрытие доходов и расходов (ЗакрытиеМесяца)
```
Ежемесячно 1С выполняет:
  Дт 6010..6xxx / Кт 5610  — перенос всех доходов в P&L
  Дт 5610 / Кт 7010..7xxx  — перенос всех расходов из P&L

На конец месяца: 6xxx = 0 остаток, 7xxx = 0 остаток,
  5610 ДТ-остаток = убыток за период
  5610 КТ-остаток = прибыль за период

ЕСЛИ 6xxx или 7xxx НЕ закрыты → ЗакрытиеМесяца не выполнено
```

## Аномалии в капитале и финрезультате
```
5610 остаток до ЗакрытиеМесяца ≠ 0 → нормально (нарастает в течение мес.)
5610 остаток после ЗакрытиеМесяца ≠ 0 → ошибка закрытия
5510 < 0 → накопленный убыток → проверить достаточность капитала
3040 КТ > 0 без выплаты → объявлены дивиденды, не выплачены (обязательство)
6030/6020 > 20% выручки → аномально высокие возвраты/скидки
```
