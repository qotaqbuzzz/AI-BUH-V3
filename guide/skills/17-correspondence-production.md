## Аномалии в производственных счетах
```
8110 КТ-остаток → больше оприходовано чем затрат = ошибка оценки
8112 > 0 в декабре (яровые) → урожай не оприходован → CRITICAL
8410 не закрыт → накладные не распределены → ЗакрытиеМесяца не выполнено
1320 КТ-остаток → продали больше чем произвели = минус остаток = ошибка
```
