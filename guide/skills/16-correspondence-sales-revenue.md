# SKILL: Корреспонденция — Продажи и Выручка

**Trigger:** счета 6010/6020/6030/1210/7010/3130/3510 в проводке при реализации

## Реализация товаров/продукции (полный цикл)
```
Шаг 1 — Признать выручку (отгрузка):
  Дт 1210  / Кт 6010   — выручка нетто (без НДС)
  Дт 1210  / Кт 3130   — НДС начисленный (12% от нетто)
  Итог: 1210 = сумма с НДС, 6010 = нетто, 3130 = НДС 12%

Шаг 2 — Списать себестоимость:
  Дт 7010  / Кт 1320   — себестоимость готовой продукции
  или
  Дт 7010  / Кт 1330   — себестоимость товаров

Шаг 3 — Покупатель оплачивает:
  Дт 1030  / Кт 1210   — деньги поступили

Шаг 4 — Уплата НДС в бюджет:
  Дт 3130  / Кт 1030   — НДС перечислен
```

## НДС при реализации — детали
| Дт | Кт | Операция |
|---|---|---|
| 1210 | 3130 | НДС начисленный (12% от нетто) |
| 3130 | 1420 | Зачёт входящего НДС против исходящего |
| 3130 | 1030 | Уплата НДС нетто в бюджет |

```
НДС начисленный = выручка нетто (6010) × 12%
НДС к зачёту    = закупки с НДС × 12/112
Нетто к уплате  = 3130 – 1420

Ошибка #1: использовать 12/112 для начисления (надо × 12%)
Ошибка #2: не начислять 3130 = нет ЭСФ → штраф 50% суммы
```

## Аномалии в счетах продаж
```
6010 ДТ-остаток (не сторно) → выручка "ушла в минус" = ошибка
7010 без корреспонд. 1320/1330 → себестоимость без остатка = минус запас
1210 ДТ-остаток растёт быстро → покупатели не платят → риск обесценения
3130 КТ-остаток нарастает → НДС не уплачивается → налоговая задолженность
```
